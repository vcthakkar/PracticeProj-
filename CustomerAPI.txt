Customer API
Create Asp .Net Web API that allows CRUD operation on a Customer entity. 
The focus will be on the structure of the application and not so much on more number of 
working Endpoints. 
N-tire application is desired, so do not jam all the things into single VisualStudio Project

Customer - done
Id
FirstName
LastName
Email
MemberSince

Basic Features
CRUD REST endpoints on Customer Entity - done
Connectivity with SQL Server - done
Make use of EntitFramework [Code First preferred] - done
Basic Model Validation

Advance Features
Enable JWT
Integrate Asp .Net Identity
Do not use BoilerPlate and build OWIN based pipeline
Enable CORS - done
Enable Swagger UI - done
